The proof of the while statement (while b do C) simulation, i.e. (while b do C, σp) ≼ (while b do C, σp'), is parted into two cases:
  • The label of b, i.e. lb, is less than the state equivalence label L, i.e. lb ⊑ L.
    This case is proved by lemma WellType_imply_TSim_While_labelleq_lb_l, i.e. lb ⊑ L -> (while b do C, σp) ≼ (while b do C, σp').
    In this case, the values of b under two equivalent states are always equal. So the executions from two equivalent states of the while statement always choose the same path.
    Since the lemma WellType_imply_TSim_While_labelleq_lb_l cannot be proved directly by the coinduction, we need to prove a more generalized version.
    lb ⊑ L ->
    (C1, σp) ≼ (C1',σp') ->
    (forall σp1 σp1', σp1 ≈ σp1' => (C,σp1) ≼ (C',σp1')) ->
    (C1; while b do C, σp) ≼ (C1'; while b do C', σp').
    This lemma (WellType_imply_TSim_While_labelleq_lb_l_aux) can be proved directly by coinduction.
  • The label of b is not less than the state equivalence label L, i.e. lb ⋢ L.
    This case is proved by lemma WellType_imply_TSim_While_nlabelleq_lb_l, i.e. lb ⋢ L -> (while b do C, σp) ≼ (while b do C, σp').
    Again, this lemma cannot be proved by the coinduction directly. Because (while b do C, σp) can be executed to be the form (ASkip, σp''), (C'; while b do C, σp'') or (while b do C, σp''). The three cases should be all considered. So an auxiliary lemma called WellType_imply_TSim_While_aux is introduced.
    In the proof of WellType_imply_TSim_While_aux, one fact should be mentioned: lb ⋢ L -> ASkip ≼ (while b do C, σp) . It is proved by lemma TSim_Skip_While. This lemma is right, because the following two facts:
    • lb ⋢ L -> (while b do C, (σp, σs)) ↪* (ASkip, (σp', σs')) -> σp ≈ σp'.
    • lb ⋢ L -> (while b do C, (σp, σs)) ↪* (ASkip, (σp', σs')) -> σs ≈ σs'.
    They are proved by lemma starparaSeqStep_While_Skip_PStateApprox_While0 and lemma starSeqStep_Skip_SStateApprox_while separately.

Lemma WellType_imply_TSim_While_labelleq_lb_l:

  constructor.
  split.

      }

        eapply SeqStep_seqstateII; eauto.
      }

                          (ASeq (ASeq ac AEnd) (AWhiledo b (ASeq ac AEnd)), (ts' :: tpc', ps', sst'))). {
        ∃ (S O).
        simpl.
        eapply ex_intro.
        split; eauto.
      }
      
      split; auto.
      split; auto.
      split; auto.
      {
        eapply WellType_imply_TSim_While_labelleq_lb_l_aux; eauto.
        - intro.
          eapply SeqStep_TTerm; eauto.
        - intro.
