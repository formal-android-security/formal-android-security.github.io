Basic Definitions

Variables are represented by the type nat of natural numbers. The variables are seperated as private variables recorded in the private store, and shared variables recorded in the shared store.
Definition Vars := nat.

The Policy

A Label is a set of variables or a special element LTop representing the universal set. The set of Label forms a lattice with the least upper bound function lunion L1 L2, i.e. L1 ⊔ L2. It also forms a partially order set with the order function labelleq L1 L2, i.e. L1 ⊑ L2. LTop is the greatest element.

Inductive Label :=
| LTop
| LElemSet (les:set Vars).

Definition lunion (l1 l2:Label) :=
  match (l1,l2) with
    | (LTop,_) ⇒ LTop
    | (_,LTop) ⇒ LTop
    | (LElemSet les1, LElemSet les2) ⇒ LElemSet (set_union Nat.eq_dec les1 les2)
  end.

Notation "L1 '⊔' L2" := (lunion L1 L2) (at level 80, right associativity).

Definition labelleq (l1 l2:Label) :=
  l2 = LTop ∨
  (∃ les1 les2, l1 = LElemSet les1
                     ∧l2 = LElemSet les2
                     ∧ incl les1 les2
  ).

Notation "L1 '⊑' L2" := (labelleq L1 L2) (at level 80, right associativity).

Definition labeleq (l1 l2:Label) :=
  labelleq l1 l2 ∧ labelleq l2 l1.

A Policy is pair of labels. The first element represents the access permission, and the second element represents the send permission. The access permission is the set of shared variables a process can visit, and the send permisison is the set of shared variables a process can send out of the device.
Definition Policy := (Label× Label)%type.

The Language

Programming Language

The arithmetical expression and boolean expression only refer the private variables.
Inductive Expr :=
| EVar (x:Vars)
| ENum (n:nat)
| EAdd (e1:Expr) (e2:Expr).

Inductive Bexp :=
| Btrue
| Bfalse
| Beq (e1:Expr) (e2:Expr)
| Bleq (e1:Expr) (e2:Expr)
| Bnot (b:Bexp).

Hsend/Hrecv and Lsend/Lrecv primitives are used to perform IPC(Inter-Process Communication). Out primitive sends the information outside of the device. Put/Get primitives access the confidential data. Other statements are standard.
Inductive Cmd :=
| Assign (x:Vars) (e:Expr)
| Hsend (e1:Expr) (e2:Expr)
| Lsend (e1:Expr) (e2:Expr)
| Hrecv (x:Vars) (y:Vars)
| Lrecv (x:Vars) (y:Vars)
| Out (e:Expr)
| Ifelse (b:Bexp) (c1:Cmd) (c2:Cmd)
| Whiledo (b:Bexp) (c:Cmd)
| Seq (c1:Cmd) (c2:Cmd)
| Get (x:Vars) (y:Vars)
| Put (x:Vars) (e:Expr)
| Skip.

Annotated commands instrument some information into the commands. The only difference between these two version commands is the Hsend primitive, i.e. Hsend(e1,e2) versus Hsend(e1,e2,L). The End is an auxiliary command to record the scope of if and while command.
Inductive AnnoCmd :=
| AAssign (x:Vars) (e:Expr)
| AHsend (e1:Expr) (e2:Expr) (l:Label)
| ALsend (e1:Expr) (e2:Expr)
| AHrecv (x:Vars) (y:Vars)
| ALrecv (x:Vars) (y:Vars)
| AOut (e:Expr)
| AIfelse (b:Bexp) (ac1:AnnoCmd) (ac2:AnnoCmd)
| AWhiledo (b:Bexp) (ac:AnnoCmd)
| ASeq (ac1:AnnoCmd) (ac2:AnnoCmd)
| AGet (x:Vars) (y:Vars)
| APut (x:Vars) (e:Expr)
| ASkip
| AEnd.

Definition Tid := nat.

Private Store only save values of private variables.

Definition Policys := Tid → Policy.

The information sent outside by any process is recorded in OutState
Definition OutState := list (TidSet × nat)%type.

The shared state is consist of four parts: the shared store, the high message queues, the low message queues and the out state.
Definition SharedState := (SharedStore × HMsgQueueSet × LMsgQueueSet × OutState)%type.

The state is consist of two parts: the private state, which can be accessed by only individual process and the shared state, which can be accessed by multiple processes.
Definition State := (PrivateStateSet × SharedState)%type.

Sequential Semantics

Evaluation of expressions

Definition ts_union (ts1 ts2:TidSet) := set_union eq_nat_dec ts1 ts2.

Fixpoint ExprInterp (e:Expr) (ps:PrivateStore):=
  match e with
    | EVar x ⇒ ps(x)
    | ENum n ⇒ (empty_set Tid,n)
    | EAdd e1 e2 ⇒ match (ExprInterp e1 ps,ExprInterp e2 ps) with
                      | ((ts1,n1),(ts2,n2)) ⇒ (ts_union ts1 ts2, n1+n2)
                    end
  end.

Fixpoint BexpInterp (b:Bexp) (ps:PrivateStore) :=
  match b with
    | Btrue ⇒ (empty_set Tid,true)
    | Bfalse ⇒ (empty_set Tid,false)
    | Beq e1 e2 ⇒ match (ExprInterp e1 ps ,ExprInterp e2 ps) with
                     | ((ts1,n1), (ts2,n2)) ⇒ (ts_union ts1 ts2, beq_nat n1 n2)
                   end
    | Bleq e1 e2 ⇒ match (ExprInterp e1 ps ,ExprInterp e2 ps) with
                      | ((ts1,n1), (ts2,n2)) ⇒ (ts_union ts1 ts2, Arith.Compare_dec.leb n1 n2)
                    end
    | Bnot b1 ⇒ match (BexpInterp b1 ps) with
                   | (ts,bvalue) ⇒ (ts,negb bvalue)
                 end
  end.

Definition updateps (ps:PrivateStore) (x:Vars) (v:(TidSet× nat)) :=
  fun (y:Vars) ⇒
    if (eq_nat_dec x y)
    then v
    else ps(y).

Definition updatepsts (psts:PrivateStateSet) (i:Tid) (v: PrivateState) :=
  PSTSMod.set psts i v.

Definition updatess (ss:SharedStore) (x:Vars) (n:(TidSet× nat)) :=
  fun y ⇒ if (eq_nat_dec x y)
           then n
           else ss(y).

Fixpoint tpc_union (tpc: TPC) : TidSet:=
  match tpc with
    | nil ⇒ empty_set nat
    | a::tpc' ⇒ set_union eq_nat_dec a (tpc_union tpc')
  end.

Definition ts_eq (ts1 ts2:TidSet) := ts1 = ts2.

Definition ts_add (t :Tid) (ts2 :TidSet) := set_add eq_nat_dec t ts2.

Definition tid_in (t:Tid) (ts:TidSet) := set_mem eq_nat_dec t ts.

Definition gap (p:Policy) :=
  match p with
    | (gap,_) ⇒ gap
  end.

Definition gsp (p:Policy) :=
  match p with
    | (_,gsp) ⇒ gsp
  end.

Transition semantics for annotated commands

Hsend/Hrecv and Lsend/Lrecv primitives are non-blocking operations. Hsend/Lsend fail when the target process is not exist. Hrecv/Lrecv fail when its high message queue/low message queue are empty.
Inductive SeqStep : Tid → Policys → SeqConfig → SeqConfig → Prop :=
  SSSkip : ∀ seqstate t rho,
             SeqStep t rho (ASkip,seqstate) (ASkip,seqstate)
| SSAssign : ∀ ts n x e tpc t rho ps ps' sst,
                ExprInterp e ps = (ts,n) →
                ps' = updateps ps x (ts_add t (ts_union ts (tpc_union tpc)),n) →
                SeqStep t rho
                        ((AAssign x e), ((tpc,ps), sst))
                        (ASkip, ((tpc,ps'), sst))
| SSHsendsucc : ∀ e1 tsi i e2 ts n l rho hqi tpc hqi' ss hqs lqs ps alpha poi pot hqs' (sst sst':SharedState) (pst:PrivateState) t,
                  pst = (tpc, ps) →
                  ExprInterp e1 ps = (tsi,i) →
                  ExprInterp e2 ps = (ts,n) →
                  sst = (ss,hqs,lqs,alpha) →
                  Some hqi = HMSGSMod.get hqs i →
                  hqi' = hqi ++ ((t,ts_union ts (tpc_union tpc),n)::nil) →
                  hqs' = HMSGSMod.set hqs i hqi' →
                  sst' = (ss,hqs',lqs,alpha) →
                  poi = rho(i) →
                  pot = rho(t) →
                  labelleq l (gap poi) →
                  (labelleq (gsp poi) (gsp pot)) →
                  SeqStep t rho
                    ((AHsend e1 e2 l), (pst, sst))
                    (ASkip, (pst, sst'))
| SSHsendfail : ∀ e1 tsi i e2 l rho tpc ss hqs lqs ps alpha sst pst t,
                  pst = (tpc, ps) →
                  ExprInterp e1 ps = (tsi,i) →
                  sst = (ss,hqs,lqs,alpha) →
                  None = HMSGSMod.get hqs i →
                  SeqStep t rho
                    ((AHsend e1 e2 l), (pst, sst)) (ASkip, (pst, sst))
| SSLsendsucc : ∀ e1 tsi i e2 ts n rho lqi tpc lqi' ss hqs lqs ps alpha poi pot lqs' (sst sst':SharedState) (pst:PrivateState) t,
                  pst = (tpc, ps) →
                  ExprInterp e1 ps = (tsi,i) →
                  ExprInterp e2 ps = (ts,n) →
                  sst = (ss,hqs,lqs,alpha) →
                  Some lqi = LMSGSMod.get lqs i →
                  lqi' = lqi ++ ((t,ts_union ts (tpc_union tpc),n)::nil) →
                  lqs' = LMSGSMod.set lqs i lqi' →
                  sst' = (ss,hqs,lqs',alpha) →
                  poi = rho i →
                  pot = rho t →
                  (labelleq (gsp poi) (gsp pot) ) →
                  SeqStep t rho
                    ((ALsend e1 e2), (pst, sst))
                    (ASkip, (pst, sst'))
| SSLsendfail : ∀ e1 tsi i e2 rho tpc ss hqs lqs ps alpha sst pst t,
                  pst = (tpc, ps) →
                  ExprInterp e1 ps = (tsi,i) →
                  sst = (ss,hqs,lqs,alpha) →
                  None = LMSGSMod.get lqs i →
                  SeqStep t rho
                    ((ALsend e1 e2), (pst, sst)) (ASkip, (pst, sst))
| SSHrecvsucc : ∀ t tpc pst i ts n hqt' pst' rho ss hqs hqs' lqs alpha x y sst sst' ps ps',
                  pst = (tpc, ps) →
                  sst = (ss, hqs, lqs, alpha) →
                  HMSGSMod.get hqs t = Some ((i, ts, n)::hqt') →
                  ps' = updateps (updateps ps x (ts_add t (tpc_union tpc),i)) y (ts_add t (ts_union ts (tpc_union tpc)),n) →
                  pst' = (tpc, ps') →
                  hqs' = HMSGSMod.set hqs t hqt' →
                  sst' = (ss, hqs', lqs, alpha) →
                  SeqStep t rho
                    ((AHrecv x y), (pst, sst))
                    (ASkip, (pst', sst'))
| SSHrecvfail : ∀ t (pst:PrivateState) rho ss hqs lqs alpha pst' x y ps ps' sst tpc ,
                  pst = (tpc, ps) →
                  sst = (ss, hqs, lqs, alpha) →
                  HMSGSMod.get hqs t = Some nil →
                  ps' = updateps (updateps ps x (ts_add t (tpc_union tpc),0)) y (ts_add t (tpc_union tpc),0) →
                  
                  pst' = (tpc, ps') →
                  SeqStep t rho
                    ((AHrecv x y), (pst, sst))
                    (ASkip, (pst', sst))
| SSLrecvsucc : ∀ t tpc (pst pst':PrivateState) i ts n rho ss hqs lqs alpha lqs' lqt' x y (sst sst':SharedState) ps ps',
                  pst = (tpc, ps) →
                  sst = (ss, hqs, lqs, alpha) →
                  LMSGSMod.get lqs t = Some ((i, ts, n)::lqt') →
                  ps' = updateps (updateps ps x (ts_add t (tpc_union tpc),i)) y (ts_add t (ts_union ts (tpc_union tpc)),n) →
                  pst' = (tpc, ps') →
                  lqs' = LMSGSMod.set lqs t lqt' →
                  sst' = (ss, hqs, lqs', alpha) →
                  SeqStep t rho
                    ((ALrecv x y), (pst, sst))
                    (ASkip, (pst', sst'))
| SSLrecvfail : ∀ t tpc pst rho ss hqs lqs alpha pst' x y sst ps ps',
                  pst = (tpc, ps) →
                  sst = (ss, hqs, lqs, alpha) →
                  LMSGSMod.get lqs t = Some nil →
                  ps' = updateps (updateps ps x (ts_add t (tpc_union tpc),0)) y (ts_add t (tpc_union tpc),0) →
                  
                  pst' = (tpc, ps') →
                  SeqStep t rho
                    ((ALrecv x y), (pst, sst))
                    ((ASkip), (pst', sst))
| SSGet : ∀ ss y n x tpc t rho ps hqs lqs alpha ps' ts pst pst' sst,
            pst = (tpc,ps) →
            sst = (ss,hqs,lqs,alpha) →
            ss y = (ts, n) →
            ps' = updateps ps x (ts_add t (tpc_union tpc), n) →
            pst' = (tpc, ps') →
            SeqStep t rho
              ((AGet x y), (pst, sst))
              (ASkip, (pst', sst))
| SSPut : ∀ e ps ts n ss x tpc t rho hqs lqs alpha ss' pst sst sst',
            pst = (tpc, ps) →
            sst = (ss, hqs, lqs, alpha) →
            ExprInterp e ps = (ts,n) →
            ss' = updatess ss x (ts_union ts (tpc_union tpc), n) →
            sst' = (ss', hqs, lqs, alpha) →
            SeqStep t rho
              ((APut x e), (pst, sst))
              ((ASkip), (pst, sst'))
| SSOut : ∀ e ts n tpc t rho ss ps hqs lqs alpha alpha' (pst:PrivateState) sst,
            pst = (tpc, ps) →
            sst = (ss, hqs, lqs, alpha) →
            ExprInterp e ps = (ts,n) →
            alpha' = (ts_add t (ts_union ts (tpc_union tpc)), n) :: alpha →
            SeqStep t rho
              ((AOut e), (pst, sst))
              (ASkip, (pst, (ss, hqs, lqs, alpha')))
| SSSeq : ∀ ac1 seqstate ac1' seqstate' ac2 t rho,
            SeqStep t rho (ac1, seqstate) (ac1', seqstate') →
            SeqStep t rho ((ASeq ac1 ac2), seqstate) ((ASeq ac1' ac2), seqstate')
| SSSeqSkip : ∀ ac seqstate t rho,
                SeqStep t rho ((ASeq ASkip ac), seqstate) (ac, seqstate)
| SSIfelsetrue : ∀ sst tpc t rho b ps ts ac1 ac2 pst pst',
                   pst = (tpc,ps) →
                   BexpInterp b ps = (ts, true) →
                   pst' = (ts::tpc,ps) →
                   SeqStep t rho
                           ((AIfelse b ac1 ac2),(pst, sst))
                           (ac1, (pst', sst))
| SSIfelsefalse : ∀ sst tpc t rho b ps ts ac1 ac2 pst pst',
                   pst = (tpc,ps) →
                   BexpInterp b ps = (ts, false) →
                   pst' = (ts::tpc,ps) →
                   SeqStep t rho
                           ((AIfelse b ac1 ac2),(pst, sst))
                           (ac2, (pst', sst))
| SSWhiledotrue : ∀ sst tpc t rho b ps ts ac pst pst',
                   pst = (tpc,ps) →
                   BexpInterp b ps = (ts, true) →
                   pst' = (ts::tpc,ps) →
                   SeqStep t rho
                           ((AWhiledo b ac),(pst, sst)) ((ASeq ac (AWhiledo b ac)),(pst', sst))
| SSWhiledofalse : ∀ tpc t rho b ps ts ac pst sst,
                   pst = (tpc,ps) →
                   BexpInterp b ps = (ts,false) →
                   SeqStep t rho
                           ((AWhiledo b ac),(pst, sst))
                           (ASkip, (pst, sst))
| SSEnd: ∀ tpc ts t rho ps pst sst pst',
                   pst = (ts::tpc,ps) →
                   pst' = (tpc,ps) →
                   SeqStep t rho
                           (AEnd, (pst, sst))
                           (ASkip, (pst', sst)).

Function kSeqStep (k:nat) (t: Tid) (rho: Policys) (sc:SeqConfig) (sc':SeqConfig): Prop :=
  match k with
    | S n ⇒ ∃ sc'', SeqStep t rho sc sc'' ∧ kSeqStep n t rho sc'' sc'
    | O ⇒ sc = sc'
  end.

Function starSeqStep (t: Tid) (rho: Policys) (sc:SeqConfig) (sc':SeqConfig) : Prop :=
  ∃ k, kSeqStep k t rho sc sc'.

Transition semantics for annotated program

An annotated program may consist of multiple processes, within each executing an annotated command.

Inductive ParaStep : Policys → ParaConfig → ParaConfig → Prop :=
| PStep : ∀ psts rho pe pe' psts' aci aci' i sst sst' pst pst',
            PSTSMod.get psts i = Some pst →
            PEMod.get pe i = Some aci →
            SeqStep i rho
              (aci, (pst, sst)) (aci', (pst', sst')) →
            pe' = updatepe pe i aci' →
            psts' = updatepsts psts i pst' →
            ParaStep rho
              (pe,(psts, sst)) (pe',(psts', sst')) .

Function kParaStep (k:nat) (rho: Policys) (t:ParaConfig) (t':ParaConfig): Prop :=
  match k with
    | S n ⇒ ∃ t'', ParaStep rho t t'' ∧ kParaStep n rho t'' t'
    | O ⇒ t = t'
  end.

Function starParaStep (rho: Policys) (t:ParaConfig) (t':ParaConfig) : Prop :=
  ∃ k, kParaStep k rho t t'.
